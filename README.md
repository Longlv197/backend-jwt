# Hướng dẫn cài đặt

1. git clone https://github.com/Longlv197/backend-jwt.git
2. composer install
3. cp .env.example .env
4. php artisan key:generate
5. setup database in .env
6. php artisan migrate --seed
7. php artisan serve
8. The site will run localhost:8000
